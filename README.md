# Brain Tumor Classification using MobileNetV2
## Overview

This project is a Machine Learning-based web application using Streamlit for classifying brain tumor types from MRI images. The model can detect whether a brain MRI image contains a tumor and classify it into one of the predefined categories.

The system classifies images into 4 classes:
- Glioma Tumor
- Meningioma Tumor
- Pituitary Tumor
- No Tumor

This project aims to assist in early detection of brain tumors using deep learning techniques.

## Dataset
The dataset used in this project is publicly available at [Brain Tumor MRI Dataset](https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri) of the Kaggle platform
